Para compilar:

Se não existir nenhuma pasta "bin" no mesmo lugar que o makefile
mkdir bin

Instalar g++
*ou*
Instalar llvm e clang, e trocar g++ por clang++ no makefile

Então usar o comando make, o arquivo executável estará na pasta bin
